# Tech Stack & Architecture

## Stack

- Next.js 16 (App Router, Server Components, TypeScript strict, Turbopack by default)
- React 19 + TypeScript (strict: true, no any types ever)
- Tailwind CSS v4 + shadcn/ui (New York style, neutral base)
- Supabase Pro (PostgreSQL, Auth, Storage, Realtime)
- Zustand (client state), React Hook Form + Zod (forms)
- Recharts (charts), date-fns (dates), PapaParse (CSV), Lucide React (icons)
- Resend (email), Razorpay (payments), @react-pdf/renderer (PDF)
- next-sitemap (SEO), Tiptap (rich text editor)

## Architecture Rules (NON-NEGOTIABLE)

1. ALL business logic in src/lib/engines/ ‚Äî NEVER in Supabase functions
2. Supabase ONLY for: database, auth, storage, realtime
3. API routes are thin controllers calling engine functions
4. Server Components for SEO pages, Client Components only for interactivity
5. RLS enforces multi-tenancy at database level
6. Every page must have SEO metadata + structured data (JSON-LD)
7. Prices stored as integers in Rupees, displayed as ‚Çπ1,999 (Indian numbering)
8. Mobile-first responsive design (320px to 2560px)

## Next.js 16 Conventions (BREAKING CHANGES ‚Äî MUST FOLLOW)

- **Async Request APIs**: `cookies()`, `headers()`, `draftMode()` are async ‚Äî always `await` them. `params` and `searchParams` in pages/layouts/routes are Promises ‚Äî `await` before use.
- **Proxy not Middleware**: the root file is `proxy.ts` (NOT `middleware.ts`) and exports a `proxy` function. Runtime is `nodejs` only (no edge runtime).
- **next.config.ts**: `turbopack` is a top-level option (not under `experimental`). Use `images.remotePatterns` (NOT the deprecated `images.domains`).
- **Tailwind v4**: CSS-first config via `@theme` in globals.css. A `tailwind.config.ts` is wired via `@config` for brand tokens.
- **Turbopack** is the default bundler for `next dev` and `next build` (no `--turbopack` flag).
- Use typed route helpers `PageProps<'/route'>`, `LayoutProps<'/route'>`, `RouteContext<'/route'>` generated by `next typegen`.

## Folder Convention

- src/app/(marketing)/ ‚Äî Public pages with shared header/footer
- src/app/(auth)/ ‚Äî Login, register, forgot password
- src/app/(dashboard)/ ‚Äî Client portal (protected by auth middleware)
- src/app/(admin)/ ‚Äî Internal admin panel (super_admin role only)
- src/lib/engines/ ‚Äî Business logic engines (recommendation, memory, pricing, occasions, archetype, narrative, gift-builder, approval)
- src/lib/services/ ‚Äî External integrations (razorpay, resend, storage)
- src/lib/supabase/ ‚Äî Supabase clients (browser, server, admin)
- src/components/ organized by: ui/, brand/, marketing/, catalog/, gift-builder/, quote/, dashboard/, admin/, shared/
- src/data/ ‚Äî Static data and seed content
- src/emails/ ‚Äî React Email templates for Resend
